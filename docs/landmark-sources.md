# Colchester Landmark Sources

This project seeds landmark names and accessibility cues using publicly available references:

- University of Essex Colchester campus overview: https://www.essex.ac.uk/life/colchester-campus
- University of Essex facilities pages: https://www.essex.ac.uk/life/colchester-campus/facilities
- University of Essex study locations (library, Silberrad, corridors): https://www.essex.ac.uk/student/places-to-study/study-places-colchester
- University of Essex accessibility and lifts list: https://www.essex.ac.uk/visit-us/plan-your-visit/access-information
- OpenStreetMap base geodata (map contributors): https://www.openstreetmap.org

Notes:
- The initial JSON is intentionally human-readable and can be expanded incrementally.
- For production-grade coordinate precision, validate landmark points against current OpenStreetMap geometry and in-person campus surveys.
